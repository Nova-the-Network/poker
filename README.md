## About

Bienvenue sur Nova Poker, un jeu de poker en ligne développé par Nova Network.

## Features

- Gestion complète des parties de poker
- API REST pour les interactions
- Système de génération de QR codes
- Interface utilisateur intuitive
- Support multi-joueurs

## Installation

### Prérequis
- PHP 7.4 ou supérieur
- Serveur web (Apache/Nginx)
- MySQL/MariaDB

### Étapes

1. Cloner le dépôt
2. Importer la base de données
3. Configurer les fichiers

## Utilisation

- poker.php pour lancer une nouvelle partie
- poker_api.php pour les interactions programmatique
